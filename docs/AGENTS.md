# Instructions for AI Agents and Assistants

This repository uses a monorepo structure. All documentation is managed under the `/docs` directory. To optimize token usage, maintain consistency, and prevent recurring issues, you **must** strictly adhere to the following rules before, during, and after your tasks.

## Review the FAQ First
*   **Action**: Read the [FAQ](./faq.md) completely before starting any development or modification tasks.
*   **Purpose**: Prevent rewriting existing solutions and understand known constraints.

## Update the FAQ
*   **Action**: If you encounter any questions or blockers during your task, and subsequently find a resolution, you **must** document the question and its answer in the [FAQ](./faq.md) immediately.
*   **Purpose**: Build a shared knowledge base and save tokens for future reasoning.

## Track Documentation Changes
*   **Action**: Whenever you modify any files matching `/docs/*.md`, you **must** append the details to [HISTORY](./history.md).
*   **Required Content**: Clearly state **what** you changed and the explicit **reason why** the change was necessary.

## Log Unresolved Issues
*   **Action**: If you identify any missing considerations, contradictions, unresolved problems, open questions, or remaining tasks, immediately add them to [TODO](./TODO.md).
*   **Format**: Use concise, actionable bullet points.
